# Dart 語法篇
基本的組成「變數，字串，數字」

## chapter02-1 homework
Q1 : 有一個文字表達的變數，其變數名為 jobInfo，其內容叫做全端工程師   
Q2 : 有一個文字表達的變數，其變數名為 description，其內容叫做「會寫給用戶看的，也會寫接觸資料 庫的」

Q4 : 將這些內容組合成一個大型字串，並使用 print 印在畫面上
